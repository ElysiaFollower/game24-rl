# Use the course-specified model as the primary model

We will use `Qwen2.5-1.5B-Instruct` as the primary model from the start, rather than first reproducing the LLM4Game24 path on `Qwen2.5-0.5B`. The project prioritizes obtaining a strong course-relevant visible score quickly; using the required model early gives more time for tuning the actual deliverable and avoids spending early cycles on a smaller smoke-test model whose result would not be the main claim.
